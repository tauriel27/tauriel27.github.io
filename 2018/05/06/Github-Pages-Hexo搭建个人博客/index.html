Github Pages+Hexo搭建个人博客

Posted on 2018-05-06 | Edited on 2018-05-15 | In Web技术

Github Pages 是面向用户、组织和项目开放的公共静态页面搭建托管服 务，站点可以被免费托管在 Github 上，你可以选择使用 Github Pages 默 认提供的域名 github.io 或者自定义域名来发布站点。

Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Heroku上，是搭建博客的首选框架。

let’s do it !!!

环境（macOS）

  • Github 账号
  • Git
  • Node.js
  • npm
  • Hexo

Github

注册Github && 安装git

$ brew install git

Create a new repository

username 和用户名相同

安装Node.js

  1. 下载 Node.js 直接安装
  2. 使用 nvm（Node.js版本管理器）安装
# install nvm
$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash

# set environment
$ export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh" # This loads nvm

# install Node.js v8.11.1 LTS
$ nvm install 8.11.1

# check Node.js & npm version
$ node -v
$ npm -v

安装Hexo

# install nvm
$ sudo npm install -g hexo

# check version
$ hexo -v

Hexo博客

# initialization，username 改成自己的
$ hexo init tauriel27.github.io

# configure your blog
$ cd ~/tauriel27.github.io

# get 'next' theme
$ git clone https://github.com/theme-next/hexo-theme-next themes/next

# modify _config.yml file
$ vim ~/tauriel27.github.io/_config.yml


# install dependency
$ npm install

# genarate
hexo g

# run server
hexo s

cmd 点击 http://localhost:4000/ 在浏览器打开，这就是在本机上生成的静态页面，Ctrl+c 终止程序

##部署到Github

# install deployer
$ npm install hexo-deployer-git –save

# modify _config.yml


# deploy
$ hexo d

浏览器打开 tauriel27.github.io 就可以看到自己的博客了

===========================
发布新博客的时候记得先clean一下

$ hexo clean
$ hexo generate
$ hexo depoly

快捷键

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
